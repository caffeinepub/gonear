# Specification

## Summary
**Goal:** Build GoNear, a job exchange platform where employees can find colleagues at the same company who want to swap job locations, manage swap requests, message matches, and provide admin oversight.

**Planned changes:**
- Implement Internet Identity authentication for secure sign up and login
- Create user profiles with name, company, job role, current city, preferred city, salary range, and experience
- Build matching algorithm that finds users wanting to swap cities within same company and role
- Implement swap request system with send, accept, and reject functionality
- Add in-app messaging between matched users
- Create admin panel for viewing all users, verifying profiles, and approving/blocking users
- Apply clean, mobile-friendly design with blue as primary color

**User-visible outcome:** Users can sign up, create profiles, discover matching colleagues who want to swap locations, send and manage swap requests, message potential swap partners, and admins can oversee and verify the platform users.
